# Football_data_analysis
This is a Data Science project, where I analyze football player statistics with machine learning solutions.

This project explores player performance data using data science and machine learning techniques to investigate and evaluate two statistical hypotheses related to player performance.

The workflow includes data collection, cleaning, preprocessing, and exploratory data analysis (EDA) to understand patterns and relationships within the dataset. Various data visualization techniques are used to highlight trends and support hypothesis exploration.

After preparing the dataset, statistical analysis and machine learning methods are applied to test the hypotheses and identify meaningful correlations between player attributes and performance metrics. The project demonstrates a complete data science pipeline, from raw data processing to model-based analysis and interpretation of results.
